## 评论钩子（自问自答置顶）

> Q：什么是 Claude / 怎么用？
> A：去 claude.com/claude-code 装个免费版（5 分钟）→ 复制我 GitHub 上的命令 → 完事。我评论区可以一对一答。

> Q：会上传我的周报吗？
> A：完全不会。整个 Obao 是个本地 skill，下属名和 OKR 不会离开你电脑。

> Q：能不能不用 Claude？
> A：暂时只支持 Claude Code。后面看反馈考虑接 ChatGPT / Cursor。

## 配套：要不要做个落地页？

Pet Y 做了 pety.online，把"使用门槛"从 GitHub 降到"复制提示词"。

要不要也做一个 `obao.online`？预算 ¥80/年 · 内容 = 一页静态 HTML，含：
- 复制安装命令按钮
- 复制示例自我介绍按钮
- 复制示例周报按钮
- 链向 GitHub

成本极低、转化率成倍提升。

后期再做也行 —— 第一条小红书先 GitHub 链接顶着发。
